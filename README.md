Conteúdo feito em aula

Este programa possui a classe Carro, que te permite criar um objeto Carro e definir atributos como:
	- Modelo;
	- Cor;
	- velocidadeAtual;
	- estaLigado;
E seus métodos:
	- ligarCarro()
		que verifica se o carro já está ligado, caso contrário liga o carro.
	- desligarCarro()
		que verifica se o carro já está desligado, caso contrário desliga o carro.
	- pintarCarro(novaCor)
		que dada uma nova cor, muda a cor do carro.
